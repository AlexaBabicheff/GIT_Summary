1. Как скопировать на свой компьютер репозиторий из github.com:

- Клонирование репозитория осуществляется командой git clone <url>.  Где <url> - адрес репозитория. Если при этом надо, чтобы склонированный репозиторий имел иное название, надо после <url> вписать свое название.

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com:

- Сначала создать папку проекта на компьютере через команду mkdir <адрес на компьтере>; перейти в него с помощью cd /имя деректория и создать файл с помощью > либо команды touch; Из папки путем нажатия правой клавищи мыши открыть GitBush, и выполнить команды git init, git status, git add <folder> и git commit –m «коммент».
- далее надо создать проект на самом GitHub, взять его адрес и в GitBush ввести его с помощью команды git remote add origin <key SSH>, затем выполнить команду git push origin main

3. Как получить на локальный компьютер изменения с github.com:

- команда git pull

4. Какую информацию нам дают команда : git status git log git remote -v 

- git status показывает есть ли изменения у файла
- git remote –v покаызвает привязку к репозиторию на GitHub
- git log показывает прописанный коммит

5. Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory):
	- git checkout < >


6. Как вернуть файл из индекса в рабочий каталог?
 
- git restore --staged

7. Как отменить ошибочный commit ?
 Если коммит еще не выполнен - команда git reset --hard HEAD, либо git reset --hard HEAD~(количество шагов назад) 


8. Как исправить коммит отправленный на сервер GitHub?

- git add.
- git commit --amend
